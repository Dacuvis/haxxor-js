# 👾 HaxxorJS

A minimalist, blazing-fast, and lightweight web framework built on top of the **Bun** runtime. Inspired by the genius hacking skills and cyber-punk style of Haxxor Bunny from *Honkai Impact 3rd*, HaxxorJS is designed to be sleek, modular, and resource-friendly while delivering raw, uncompromised performance.

## ✨ Features

- **🚀 Bun-Native:** Built from the ground up leveraging ultra-fast `Bun.serve`.
- **📍 Dynamic Routing:** Out-of-the-box support for URL parameters (e.g., `/users/:id`).
- **🔍 Query String Parsing:** Automatic handling of URL query parameters natively.
- **🔌 Plug-and-Play:** Simple plugin system using the `.use()` method to modularize your routes.
- **📦 Zero Heavy Dependencies:** Extremely low memory footprint and lightweight.
